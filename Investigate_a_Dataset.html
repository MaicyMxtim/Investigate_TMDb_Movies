Dataset Description¶

The dataset I have picked for this project is the 'TMDb Movie data' dataset. It is originally sourced from Kaggle and some of the original columns have been dropped. This dataset can be used in identifying what makes a film successful. It contains the following columns:

  • id
  • imdb_id
  • popularity
  • budget
  • revenue
  • original_title
  • cast
  • homepage
  • director
  • tagline
  • keywords
  • overview
  • runtime
  • genres
  • production_companies
  • release_date
  • vote_count
  • vote_average
  • release_year
  • budget_adj
  • revenue_adj

Question(s) for Analysis¶

  1. Which genres are most popular from year to year?
  2. What kinds of properties are associated with movies that have high revenues?
  3. Average Runtime Of Movies From Year To Year?
  4. Which year had the most number of movies released?
  5. Who is the most frequent actor?

Conclusions¶
